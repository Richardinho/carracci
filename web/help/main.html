Carracci Help Files

Welcome to 'Carracci', a tool for creating and editing UML class diagrams.

How to use Carracci to create UML Class diagrams

Use the editor to create and edit things by typing in commands and pressing enter. A full list of available commands is given below. Also edit diagrams by clicking and dragging objects around. Some functionality is also provided through keyboard shortcuts.

The following is a quick tutorial illustrating a typical workflow

create a diagram called 'foo' and navigate to diagram context:

create diagram foo

use diagram foo

Carracci maintains a 'context path' through the diagram which allows you to navigate through a diagram and execute commands which operate on its parts. For example, after creating a diagram, in order to create components within it it is necessary to use the 'use diagram' command as shown above to enter its context.

Create a few type boxes

create type Bar create type Blah

create a horizontal connector

create connector horizontal

click and drag on components to move them around. If the editor is in the way, use 'min' command to minimize the editor. Restore it back to its original size with 'max'

Using the command editor

type in commands at the cursor and press return. Valid commands are detailed below.

Commands for working with command editor

Listed, are commands for setting various properties for the editor environent, for example, determining it's size.

  • min : minimize this editor
  • max : maximize this editor
  • width WIDTH : set width of editor e.g. width 400
  • height HEIGHT : set height of editor e.g. height 200

Commands for creating and editing diagrams

load

load existing diagrams by name into the editor

syntax

load DIAGRAM

e.g.: load commandPattern
