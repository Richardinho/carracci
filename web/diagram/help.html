Help page

Carracci is an application for creating and editing UML class diagrams

Anatomy

there are 3 main sections within the application:

the navigation bar is the thin black rectangle at the top of the viewport spanning its width. currently, it contains only the help button

the sidebar runs along the right hand side of the viewport. it contains the options menu

the viewing pane is the largest within the application. it contains a diagram when one is loaded; otherwise it is empty

Editing a diagram

you can edit in a number of different ways: moving boxes around, adjust position of connectors, change line types, change arrow types and more

Diagram

when you first start the application, a diagram is already loaded into the viewing pane. you can delete it by clicking on delete diagram in the menu. you can now either create a new diagram, or else load one of the built-in diagrams

if there is not a diagram currently loaded, click on diagram under the create section in the menu. a pop up will appear which asks you to give your diagram a name. do this and click Ok. a new diagram will be created, although the viewing pane will still be empty as no elements have been added

Type boxes

click on type box under the create section in the menu. a type box will appear somewhere within the viewing pane

click and drag on a type box to move it around the viewing pane

double-click on a type box to bring up a pop-up for editing it.

  • edit name of type box
  • toggle between a class and an interface
  • add, edit, and delete properties
  • add, edit, and delete methods
  • add a note
  • delete the type box

Connectors

click on horizontal under the create section in the menu. a horizontal connector will appear somewhere within the viewing pane

click on vertical under the create section in the menu. a vertical connector will appear somewhere within the viewing pane

connectors, both horizontal and vertical flavours, consist of three lines drawn between 4 nodes. the nodes are invisible but can be thought of as circles existing at the meeting and end points of the three lines.

you can click on a node and drag it around. this permits the shape of the connector of which it is a part to be altered.

the node can only be moved within certain constraints. if a horizontal connector is connected to a type box, the two nodes closest the type box can only be moved vertically within the range of the top and bottom of the type box, and the node closest to the type box cannot be moved horizontally at all. similarly, for a vertical connector connected to a type box, the two nodes closest to the type box can only be moved horizontally within the range of the left and right edges of the type box, and the node closest to the type box cannot be moved vertically at all

double-click on a node to bring up a popup that asks if you want to delete the connector. click delete to delete the connector

alt-click on a node to toggle between a solid line and a dotted line

shift-alt-click on an end node to change the kind of arrow. the different types are: open arrow, closed white arrow, white diamond, black diamond, and none

shift-click on an end node that is not connected to a type box prepares it to be connected. shift-click on a type box to complete the connection (note: don't hold shift down whilst clicking on the node and the type box)

shift-click on an end node that is connected to a type box to disconnect it from the type box

Banners

click on banner under the create section in the menu. a pop up will appear allowing you to give the banner a title, an author, a creation date, and contents. click save to create the banner

double-click on the banner to bring up the pop up for further editing

Built-in diagrams

the built-in diagrams demonstrate various design patterns. they exist as an example of what is possible within this application, as a starting point for your own diagrams, and as a tool for learning about design patterns. load one of these diagrams by clicking on its name on the menu within the load section
