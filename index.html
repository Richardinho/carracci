Help

About Carracci

Carracci is an editing tool for UML class diagrams. Currently a work in progress, when complete you will be able to create classes, specify class properties and methods, show relationships such as inheritance and composition with connecting arrows, add notes, and save the diagram to either PNG or JPEG file formats.

Edit a diagram

Create and edit a uml class box

Create a class by clicking on an icon in the toolbox, or by clicking on the add button within the gui on the right hand side. A new class box will appear within the diagram canvas. Use the gui to edit class properties and methods. class boxes can be moved around the canvas simply by dragging.

Create and edit connectors

Connecting lines can be created by clicking on the icon in the toolbox. The new connector will appear in the canvas. The default has black arrows on both ends, but it is possible to change this (see below).Manipulate arrows around the canvas by clicking on the ends or the 'corner joints' of the arrow and dragging.

Delete a connector by holding down the D button and clicking on either end of the connector. A pop up will appear asking for confirmation that you want to delete the connector. Press delete to delete and cancel to cancel.

Toggle line style

Clicking and holding down the key T on top of a connector arrow toggles the connecting line from solid to dashed. This is useful when creating connecting lines signifying 'implements' relationship between classes.

Change arrow type

Clicking on a connector arrow whilst holding down the S key changes the arrow type. e.g. from black to white.

Attach lines to boxes

To connect an arrow to a box, hold down the U key, click on an arrow, then click on the box you want to connect to. The arrow will automatically reposition itself onto the box. When you now move the box, the arrow will stay attached to it. To detach an arrow from a box, hold down the U key again whilst clicking on the attached arrow.

Export a diagram

Click on the PNG icon in the toolbox to export the diagram to png format. A new window will open containing the exported image.
