Create

general command for creating all types of artifacts.

syntax

create ARTIFACT NAME | ARTIFACT_TYPE

The create command takes the following forms:

  • create diagram DIAGRAM_NAME
  • create type TYPE_NAME
  • create connector CONNECTOR_ORIENTATION
  • create property PROPERTY_NAME
  • create method METHOD_NAME
  • create arg ARG_NAME ARG_VALUE

For these commands to work the context path must be in the right state. As well as this, a diagram can only be created if there doesn't exist a current one.

connectors

Generally, the 2nd argument to the command is the name to give to the artifact. However, in the case of connectors, this argument must be one of two values: 'horizontal' for horizontally aligned connectors (meaning that their ends point left or right), or 'vertical',(meaning that their ends point up or down).

Examples

To create a type called 'Foo': create type Foo

To create a horizontally aligned connector: create connector horizontal
